# 화면 목록

## Android 앱

- 서버 접속 화면
  - 서버 주소, 룸 코드, 사용자명 입력
  - 최근 접속 프로필 선택
  - 헤드셋 연결 상태 표시

- 인터컴 패널 화면
  - 방송용 키패널 형태의 메인 화면
  - 채널별 Talk, Listen, Call 상태 표시
  - 큰 PTT 버튼
  - 키별 listen level 조절
  - 마스터 mute, speaker/listen 상태 표시

- 채널 상세 화면
  - 채널 이름
  - Talk latch 허용 여부
  - listen level
  - call signal
  - remote mic kill 수신 상태

- 오디오 설정 화면
  - 입력 장치 선택
  - 출력 장치 선택
  - 블루투스 SCO 사용 상태
  - sidetone, noise suppression, echo cancellation 설정

- 네트워크 상태 화면
  - LTE/Wi-Fi 연결 상태
  - RTT, packet loss, jitter 표시
  - 서버 재연결 상태

## PC 서버 프로그램

- 서버 대시보드
  - 서버 시작/종료
  - 로컬/외부 접속 주소
  - 현재 룸 수, 접속자 수
  - 로그 요약

- 룸 관리 화면
  - 룸 생성/삭제
  - 룸 코드/QR 표시
  - 룸별 채널 구성
  - 참가자 권한 관리

- 참가자 모니터 화면
  - 참가자별 connected, talking, listening, muted 상태
  - 입력 레벨 meter
  - packet loss, RTT
  - mute/kick/remote mic kill

- 채널 매트릭스 화면
  - 채널별 Talk/Listen 라우팅
  - partyline, group, IFB, point-to-point 타입 지정
  - 운영자 override

- 서버 설정 화면
  - signaling 포트
  - TURN/STUN 설정
  - media relay 방식
  - 로그 저장 위치
